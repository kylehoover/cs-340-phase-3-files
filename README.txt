Main class package name: group42.tickettoride.server.Communication.ServerCommunicator

To run server: java -cp app.jar;gson-2.8.0.jar group42.tickettoride.server.Communication.ServerCommunicator 8081

Package name of client app: group42.tickettoride

Emulators: It's designed for the Nexus 10 tablet, but runs best on the WXGA 10.1 tablet. It also runs well on the Pixel C, but the layout is a little stretched. We built it using API 24 Android 7.0 (Nougat).

Running the app: At the top of the login screen put the server's IP address and port number (e.g. 10.0.2.2:8081 if you're using an emulator)
